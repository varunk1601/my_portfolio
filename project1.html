Mouse Controller Using Facial Expression

A project to aid quadriplegics by using facial expressions to control a mouse cursor.

Project Overview

This project presents a Human-Computer Interaction (HCI) system that allows users to control a mouse cursor using facial expressions and eye movements. It is designed to help individuals who cannot use their hands effectively.

Features

  • Eye blink detection for clicks
  • Mouse movement based on head and facial gestures
  • Hands-free control

Technology Stack

Python, OpenCV, NumPy, Dlib, and a simple webcam.

Project Demo

This demo shows the real-time tracking of facial expressions, used to control the mouse cursor.
